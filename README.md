# General Health Query Chatbot

## Objective

The objective of this project is to build a chatbot that answers general health-related questions using a Large Language Model (LLM). The chatbot uses prompt engineering techniques and safety filters to provide helpful and responsible responses.

## Model Used

TinyLlama-1.1B-Chat-v1.0

TinyLlama was selected because it is a lightweight open-source Large Language Model suitable for conversational AI applications and can be used without paid API access.

## Technologies Used

* Python
* Hugging Face Transformers
* TinyLlama
* Google Colab
* Jupyter Notebook

## Project Features

* General health question answering
* Prompt engineering for improved responses
* Safety filter for potentially harmful medical situations
* Interactive command-line chatbot interface
* Friendly and informative responses

## Sample Questions

* What causes a sore throat?
* Is paracetamol safe for children?
* How can I prevent the flu?
* What are common symptoms of dehydration?

## Safety Measures

The chatbot does not provide medical diagnoses or emergency treatment recommendations. For serious medical concerns, users are advised to consult a qualified healthcare professional.

## Project Workflow

1. Install Required Libraries
2. Import Libraries
3. Load TinyLlama Model
4. Implement Safety Filter
5. Apply Prompt Engineering
6. Generate Responses
7. Build Chatbot Interface
8. Test Sample Queries
9. Analyze Results

## Key Findings

* Prompt engineering improved response quality and consistency.
* Safety filtering helped prevent potentially harmful outputs.
* The chatbot successfully answered common health-related questions.
* The model is suitable for educational and informational use.
